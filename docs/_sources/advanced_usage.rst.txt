Advanced CLI Usage
===================

This page explains advanced usage of the `main.py` interface, including multi-stop routing.

📌 Example:

.. code-block:: bash

   python main.py
   > Enter route: Kütüphane, Rektörlük, Geomatik Mühendisliği

✅ Result:
 - Multi-stop total distance: ~0.86 km
 - Map saved as: multi_map.html
